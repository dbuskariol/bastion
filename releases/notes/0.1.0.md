- Initial release.
- Two-Mach-O macOS menu-bar app (menu + `bastion` CLI), no privileged helper.
- `~/.ssh/config.d/bastion.conf` integration with sentinel-guarded `Include`.
- Host editor with Basic / Advanced / Raw tabs and `ssh -G` validation.
- One-click ControlMaster keepalive (8h default persist).
- In-memory shell-history import (zsh / bash / fish / known_hosts).
- Runtime detection of 10 supported terminal emulators.
- Expandable per-host diagnostics card with master uptime and attached-session count.
- ed25519 key generation; Keychain-backed passphrase storage (`kSecAttrAccessibleWhenUnlockedThisDeviceOnly`).
- Opt-in notifications for master drops, cert expiry, keychain lock.
- Sparkle 2 auto-updates with EdDSA signature verification.
